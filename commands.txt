!esportalname namn - För att registerar ett esportalnamn för din twitch-kanal. (Detta kan enbart du som streamer göra på din egna kanal)

!toggle commandnamn - stäng av eller ta på ett command som du vill.
!toggle enabled - visar vilka commands som är på.
!toggle disabled - visar vilka commands som är avstängda.

/*
 * Faceit Commands
 */
!faceit - visar streamerns faceit info

!faceit esportalnamn - visar spelarens faceit info

!faceitstats - visar streamerns faceit stats

!faceitstats esportalname - visar spelarens faceit stats.

/*
 * Esportal Commands
 */

!today - visar streamerns dagliga stats.

!today esportalnamn - visar spelarens dagliga stats.

!week - visar streamerns senaste 7 dagars stats.

!week esportalnamn - visar spelarens senaste 7 dagars stats.

!rank - visar streamerns registrerade esportalnamns rank.
      - Du kan även lägga till csgo efteråt för att få fram din csgo rank.

!rank esportalnamn - visar spelarens rank på esportal.
      - Samma sak där, du kan lägga till csgo efteråt för att få fram deras csgo rank.

!kdr - visar streamerns registrerade esportalnamns kdr.
     - Du kan även lägga till csgo efteråt för att få fram din csgo kdr.

!kdr esportalnamn - visar spelarens kdr på esportal.
     - Samma sak där, du kan lägga till csgo efteråt för att få fram deras csgo kdr.

!stats - visar streamerns generalla stats på esportal.

!stats esportalnamn - visar spelarens generalla stats på esportal.

!stat 'statistik' namn - visar statistik för den statistiken du vill ha fram.
      - Statistiker som finns är: 
      kills, deaths, assists, kdr, headshots, wins, losses, pistols, snipers, rifles, flashes, 3ks, 4ks, 2ks

!weapon nameofweapon esportalnamn - visar csgo stats för det vapnet.

!gather - visar streamerns nuvarande gatherlobby som är aktiv och länkar det till hens tittare.
	- Om streamerns har en match igång, kommer det visa mapp och score istället på matchen.

!gather esportalnamn - visar spelarens gatherlobby som är aktiv och länkar det till dina tittare.
	- Om spelaren har en match igång, kommer det visa mapp och score istället på matchen.

!match - visar streamerns nuvarande match stats, kills, deaths, assists, k/d, score mellan teams.

!match esportalnamn - visar spelarens nuvarande match stats, kills, deaths, assists, k/d, score mellan teams.

!games - visar streamerns totala games idag + elo change.

!games esportalnamn - visar spelarens totala games idag + elo change.

!gamestats - visar alla esportals gathers & games spelade under dagen och veckan.

!team - visar vilket team streamerns är med i och lite info om teamet.

!team esportalnamn - visar vilket lag spelaren är med i och lite info om teamet.



Fler kommandon kommer att komma, ge gärna lite feedback och idér vad ni vill se mer av i framtiden!
